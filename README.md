you need have vim-plug!\
Put it in C:\Users\ your username.
### Plugin
1.coc.nvim\
2.airline\
3.neadtree\
4.auto-pairs\
5.tagbar\
6.delimitmate\
7.LeaderF

### Shortcut key
gd 找到该函数定义\
,m 打开neadtree\
K 找到函数规则\
<space>n 重命名\
w= 分窗高度加\
w- 分窗高度减\
w, 分窗宽度减\
w. 分窗宽度加\
<f9> 编译运行\
,w 写入保存\
,q 写入保存并退出\
C-A 全选\
<F3> 列出当前目录文件\
[g 找到上一个错误\
g] 找到下一个错误
